## __Server__ &amp; __Client__ 개념

- __Server__ : 우리가 찾는 정보(웹페이지 문서)들을 인터넷을 통해 우리에게 제공
- __Client__ : 서버가 전달한 정보를 규칙에 따라 우리가 보기 편하게 변환
